# Linear Distribution

**Type:** `linearDistribution`  
**Category:** distribution  
**Context:** Shared — works in both DOM and canvas graphs  

Distribute points along a line from start to end.

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `count` | `float` | Count |
| `startX` | `float` | Start X |
| `startY` | `float` | Start Y |
| `endX` | `float` | End X |
| `endY` | `float` | End Y |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `points` | `any` | Points |
| `count` | `float` | Count |


## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `startX` | float | `0` | Start X |
| `startY` | float | `0` | Start Y |
| `endX` | float | `400` | End X |
| `endY` | float | `0` | End Y |
