# Path Distribution

**Type:** `pathDistribution`  
**Category:** distribution  
**Context:** Shared — works in both DOM and canvas graphs  

Distribute points along a path via arc-length sampling.

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `count` | `float` | Count |
| `path` | `path` | Path |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `points` | `any` | Points |
| `count` | `float` | Count |


## Parameters

_No configurable parameters._
