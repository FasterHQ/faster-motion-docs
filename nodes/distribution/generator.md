# Generator

**Type:** `generator`  
**Category:** distribution  
**Context:** Canvas — operates on canvas scene objects, bones, or skeletons  

Create display object clones at distribution positions. First node that creates visible shapes.

| Port | Type | Description |
|------|------|-------------|
| `distribution` | `any` | Distribution |
| `count` | `float` | Count |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `cloneIds` | `any` | Clone IDs |
| `count` | `float` | Count |
