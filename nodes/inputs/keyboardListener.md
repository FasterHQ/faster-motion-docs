# Keyboard Listener

**Type:** `keyboardListener`  
**Category:** inputs  
**Context:** DOM — operates on HTML elements via CSS selectors  

Keyboard key press/release to graph signal

## Inputs

_No inputs._

## Outputs

| Port | Type | Description |
|------|------|-------------|
| `pressed` | `float` | Pressed |


## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `key` | string | `""` | Key |
| `code` | string | `""` | Code |
| `eventType` | enum | `"keydown"` | Event. Options: `keydown`, `keyup` |
