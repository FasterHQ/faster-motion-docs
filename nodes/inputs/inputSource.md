# Input Source

**Type:** `inputSource`  
**Category:** inputs  
**Context:** DOM — operates on HTML elements via CSS selectors  

Per-scene browser input (mouse, scroll, time, keyboard). One per scene.

## Inputs

_No inputs._

## Outputs

| Port | Type | Description |
|------|------|-------------|
| `mouseX` | `float` | Mouse X (0-1) |
| `mouseY` | `float` | Mouse Y (0-1) |
| `mouseClientX` | `float` | Mouse Client X |
| `mouseClientY` | `float` | Mouse Client Y |
| `scrollX` | `float` | Scroll X (0-1) |
| `scrollY` | `float` | Scroll Y (0-1) |
| `deltaTime` | `float` | Delta Time (s) |
| `elapsed` | `float` | Elapsed (s) |
| `viewportWidth` | `float` | Viewport Width |
| `viewportHeight` | `float` | Viewport Height |
| `keysPressed` | `any` | Keys Pressed |
| `keysReleased` | `any` | Keys Released |


## Parameters

_No configurable parameters._
