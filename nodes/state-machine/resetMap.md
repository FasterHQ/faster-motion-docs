# Reset Map

**Type:** `resetMap`  
**Category:** state-machine  
**Context:** Canvas — operates on canvas scene objects, bones, or skeletons  

Applies animation reset maps during state transitions — properties animated by only one state get reset values during crossfade.

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `isTransitioning` | `float` | Is Transitioning |
| `layerDone` | `any` | Layer Done |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `applied` | `float` | Applied |


## Parameters

_No configurable parameters._
