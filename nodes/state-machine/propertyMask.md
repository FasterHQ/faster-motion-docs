# Property Mask

**Type:** `propertyMask`  
**Category:** state-machine  
**Context:** Canvas — operates on canvas scene objects, bones, or skeletons  

Build per-property animated mask from raw transforms (reset-map pattern)

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `transforms` | `any` | Raw Transforms |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `animMask` | `any` | Anim Mask |
| `propIdx` | `any` | Prop Index Map |
| `propCount` | `any` | Prop Count |


## Parameters

_No configurable parameters._
