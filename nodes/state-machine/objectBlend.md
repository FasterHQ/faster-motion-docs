# Object Blend

**Type:** `objectBlend`  
**Category:** state-machine  
**Context:** Canvas — operates on canvas scene objects, bones, or skeletons  

Blend two object pose bundles by weight (linear lerp)

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `poseA` | `attributes` | Pose A |
| `poseB` | `attributes` | Pose B |
| `weightA` | `float` | Weight A |
| `weightB` | `float` | Weight B |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `pose` | `attributes` | Pose |


## Parameters

_No configurable parameters._
