# Flip

**Type:** `flip`  
**Category:** effects  
**Context:** DOM — operates on HTML elements via CSS selectors  

FLIP layout animation — triggered on rising edge

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `trigger` | `float` | Trigger |


## Outputs

_No outputs._

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `selector` | string | `""` | Element |
| `duration` | float | `0.5` | Duration (s) (min: 0, max: 5) |
| `ease` | string | `"ease-out"` | Easing |
