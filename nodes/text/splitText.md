# Split Text

**Type:** `splitText`  
**Category:** text  
**Context:** Canvas — operates on canvas scene objects, bones, or skeletons  

Setup-only DOM text splitter — splits target element into spans (words/chars/lines).

## Inputs

_No inputs._

## Outputs

| Port | Type | Description |
|------|------|-------------|
| `count` | `float` | Count |


## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `selector` | string | `""` | Selector |
| `mode` | enum | `"words"` | Mode. Options: `words`, `chars`, `lines` |
