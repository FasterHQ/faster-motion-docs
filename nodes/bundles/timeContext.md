# Time Context

**Type:** `timeContext`  
**Category:** bundles  
**Context:** Canvas — operates on canvas scene objects, bones, or skeletons  

Exposes scheduler ambient time (deltaTime, elapsed, frame) as output ports. Clock node — feed into CycleClock or directly into a time consumer.

## Inputs

_No inputs._

## Outputs

| Port | Type | Description |
|------|------|-------------|
| `deltaTime` | `float` | Delta Time |
| `elapsed` | `float` | Elapsed |
| `frame` | `float` | Frame |


## Parameters

_No configurable parameters._
