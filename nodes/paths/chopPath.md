# Chop Path

**Type:** `chopPath`  
**Category:** paths  
**Context:** Shared — works in both DOM and canvas graphs  

Split path into N visible segments with gaps.

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `path` | `path` | Path |
| `segments` | `float` | Segments |
| `gap` | `float` | Gap |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `path` | `path` | Path |


## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `offset` | float | `0` | Offset |
