# Zig Zag

**Type:** `zigZag`  
**Category:** paths  
**Context:** Shared — works in both DOM and canvas graphs  

Alternating perpendicular peaks and valleys.

| `ridgesPerSegment` | `float` | Ridges/Segment |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `path` | `path` | Path |
