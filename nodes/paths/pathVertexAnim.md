# Path Vertex Anim

**Type:** `pathVertexAnim`  
**Category:** paths  
**Context:** Shared — works in both DOM and canvas graphs  

Animates per-vertex offsets along a path over time.

| Port | Type | Description |
|------|------|-------------|
| `path` | `path` | Path |
| `time` | `float` | Time |


## Outputs

|------|------|-------------|
| `path` | `path` | Path |
