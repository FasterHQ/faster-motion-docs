# Bone Mat4 Bundle

**Type:** `boneMat4Bundle`  
**Category:** skeleton  
**Context:** Canvas — operates on canvas scene objects, bones, or skeletons  

Gathers per-bone 2×3 world matrices from FK chain and promotes to Mat4TransformBundle for composable bone modifiers.

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `boneCount` | `float` | Bone Count |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `bundle` | `mat4Bundle` | Bone Bundle |


## Parameters

_No configurable parameters._
