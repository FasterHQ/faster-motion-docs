# For Each

**Type:** `forEach`  
**Category:** integration  
**Context:** Shared — works in both DOM and canvas graphs  

Stamp a preset per target object. Stamped nodes are read-only.

## Inputs

_No inputs._

## Outputs

| Port | Type | Description |
|------|------|-------------|
| `count` | `float` | Count |


## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `targets` | string | `[]` | Targets (object IDs) |
| `presetId` | string | `""` | Preset |
