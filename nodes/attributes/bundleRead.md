# Bundle Read

**Type:** `bundleRead`  
**Category:** attributes  
**Context:** Shared — works in both DOM and canvas graphs  

Extract a single float from an AttributeBundle by property name + index

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `bundle` | `attributes` | Bundle |


## Outputs

| Port | Type | Description |
|------|------|-------------|
| `value` | `float` | Value |


## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `propertyName` | cssProperty | `""` | Property Name |
| `index` | int | `-1` | Index |
