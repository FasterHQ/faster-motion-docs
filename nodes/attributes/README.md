# Attributes Nodes

AttributeBundle operations: read/write named attributes from bundles, extract single values by property name and index.

| Node | Type | Context | Description |
|------|------|---------|-------------|
| [Bundle Read](bundleRead.md) | `bundleRead` | shared | Extract a single float from an AttributeBundle by property name + index |
| [Attribute Read](attributeRead.md) | `attributeRead` | shared | Read a named attribute from an AttributeBundle |
| [Attribute Write](attributeWrite.md) | `attributeWrite` | shared | Stamp a named attribute onto an AttributeBundle |
