# Media Nodes

Media playback: audio tracks with RMS level output, Lottie animation control, DOM spritesheets, video sync, and WebGL video effects.

| Node | Type | Context | Description |
|------|------|---------|-------------|
| [DOM Sprite](domSprite.md) | `domSprite` | dom | Display spritesheet frame based on progress |
| [Lottie](lottie.md) | `lottie` | dom | Lottie animation — autoplay, scroll-driven, or state-controlled |
| [Video Effect](videoEffect.md) | `videoEffect` | dom | WebGL video effect — wraps VideoEffectController |
| [Audio Track](audioTrack.md) | `audioTrack` | shared | Audio playback with level output for audio-reactive animations |
| [Video Sync](videoSync.md) | `videoSync` | shared | Timeline-to-video playback synchronization. |
