# DOM Sprite

**Type:** `domSprite`  
**Category:** media  
**Context:** DOM — operates on HTML elements via CSS selectors  

Display spritesheet frame based on progress

## Inputs

| Port | Type | Description |
|------|------|-------------|
| `progress` | `float` | Progress |


## Outputs

_No outputs._

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `selector` | string | `""` | Element |
| `spritesheetSrc` | string | `""` | Spritesheet |
| `animationName` | string | `—` | Animation |
